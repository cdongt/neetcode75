

# 数据结构——树


## 什么是树


### 树的含义
有 1 个头结点, >= 2 个子节点的数据结构

### 树的特点
- 有两个子节点

### 树的分类
- 二叉树
- 非二叉树

### 树的作用
- 用于哈弗曼编码


## 怎么使用树

### 怎么创建树

```python

class TreeNode:
    def __init__(self,val,left,right):
        self.val = val
        self.left = left
        self.right = right
```
